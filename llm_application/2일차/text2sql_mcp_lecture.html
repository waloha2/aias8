03 제공되는 4가지 MCP 도구

각 도구는 단일 책임을 가지며, 순서대로 조합해 쓸 수 있습니다.

list_tables()

읽기 · 인자 없음

DB에 존재하는 모든 테이블 이름을 반환합니다. 탐색의 첫 단계.

→ "employees, products, sales"

get_schema()

읽기 · 인자 없음

전체 DDL(CREATE TABLE 문)을 반환. 컬럼, 타입, PK/FK 제약을 한 번에 파악.

→ "CREATE TABLE employees (id INTEGER PRIMARY KEY, ...)"

sample_data(table, limit=5)

읽기

특정 테이블의 샘플 행을 미리 확인. 컬럼의 실제 값 분포를 볼 때 사용.

sample_data("employees", limit=3)

query_db(sql)

SELECT 전용

임의의 SELECT 쿼리 실행. INSERT/UPDATE/DELETE/DROP은 서버에서 차단.

query_db("SELECT department, AVG(salary) FROM employees GROUP BY department")
