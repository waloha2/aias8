TF-IDF

Term Frequency - Inverse Document Frequency

0 TF-IDF란?
TF-IDF는 문서에서 단어가 얼마나 중요한지를 수치로 나타내는 방법입니다.

핵심 아이디어: 어떤 단어가 특정 문서에 자주 등장하면서, 다른 문서에는 잘 안 나오면 → 그 단어는 해당 문서에서 매우 중요합니다!
예를 들어, 4개의 문서가 있다고 합시다. 각 문서에 단어 A, B, C, D가 다음과 같이 등장합니다:

B: 10회

doc₄
A: 10회

단어 A는 모든 문서에 나옵니다 → 중요하지 않을 수 있어요!
단어 D는 doc₄에만 나옵니다 → doc₄를 대표하는 단어일 수 있어요!
1 TF (Term Frequency)
TF(w, doc) = 해당 문서에서 그 단어가 차지하는 비율

쉽게 말해: "이 문서 안에서 이 단어가 얼마나 자주 나오나?"
TF(w, doc) = count(w, doc) / |doc|
count(w, doc) = 문서에서 단어 w의 등장 횟수
|doc| = 문서의 전체 단어 수
예시: doc₁의 전체 단어 수 = 10+10+10+0 = 30
원본 (등장 횟수)
doc₁doc₂doc₃doc₄
A10101010
B1010100
C101000
D0001
⇒
TF (비율로 변환)
doc₁doc₂doc₃doc₄
TF만으로는 부족합니다. 단어 A는 TF가 높지만 모든 문서에 나오므로 변별력이 없어요!
2 DF (Document Frequency)
df(w, D) = 단어 w가 등장하는 문서의 수

쉽게 말해: "이 단어가 몇 개의 문서에 나오나?"
df(w, D) = |{doc ∈ D : w ∈ doc}|
단어등장 문서df
단어 A는 4개 문서 모두에 등장 → df가 높음 → 변별력 낮음
단어 D는 1개 문서에만 등장 → df가 낮음 → 변별력 높음
3 IDF (Inverse Document Frequency)
IDF는 DF의 역수에 로그를 취한 값입니다.

많은 문서에 나오는 단어 → IDF 낮음 (별로 중요하지 않음)
적은 문서에만 나오는 단어 → IDF 높음 (희귀하고 중요!)
IDF(w, D) = logₑ( |D| / df(w, D) )
|D| = 전체 문서 수 (여기서는 4)
단어df|D|/dfIDF = ln(|D|/df)
A: 모든 문서에 등장 → IDF = 0 (완전히 무의미!)
D: 1개 문서에만 → IDF = 1.39 (가장 희귀하고 중요!)
4 TF-IDF = TF × IDF
드디어! TF(문서 내 빈도)와 IDF(희귀도)를 곱하면
→ 특정 문서에서 그 단어가 얼마나 중요한지를 알 수 있습니다.
TF-IDF(w, doc, D) = TF(w, doc) × IDF(w, D)

핵심 해석:
- 단어 A: TF-IDF = 0 everywhere → 모든 문서에 나오니 어떤 문서도 대표하지 못함
- 단어 C: doc₁, doc₂에서 0.23 → 이 두 문서를 어느정도 구분해줌
- 단어 D: doc₄에서 0.13 → doc₄만의 고유한 특징!
5 정리
TF-IDF의 3줄 요약:
1. TF (Term Frequency)
문서 안에서 단어가 얼마나 자주 나오는지 → 빈도 비율
2. IDF (Inverse Document Frequency)
그 단어가 얼마나 희귀한지 → 희귀할수록 높은 값
3. TF-IDF = TF × IDF
자주 나오면서 + 희귀한 단어 = 그 문서의 핵심 단어!
활용: 검색 엔진, 문서 분류, 추천 시스템, 텍스트 마이닝 등에서 널리 사용됩니다.
